# Pizzeria RP V2 PRO

Structure propre pour retravailler le site.

## Fichiers

- `index.html` : structure HTML
- `css/style.css` : design complet
- `js/config.js` : URL Supabase + clé publique
- `js/supabaseClient.js` : connexion Supabase
- `js/auth.js` : connexion Discord / rôles
- `js/caisse.js` : caisse, panier, ventes
- `js/admin.js` : création / modification pizzas
- `js/stock.js` : gestion stock
- `js/staff.js` : gestion employés
- `js/compta.js` : compta + export image JPG
- `sql/installation.sql` : installation Supabase

## Installation GitHub

Uploader tous les fichiers et dossiers à la racine du repo.

## Important

Garde bien la structure :

- css/style.css
- js/*.js
- sql/installation.sql

Sinon le site ne chargera pas les fichiers.
