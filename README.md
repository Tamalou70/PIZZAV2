# Pizzeria RP V2 PRO

Structure professionnelle validée client.

## Installation GitHub

Uploader toute la structure à la racine :

- index.html
- css/style.css
- js/*.js
- sql/installation.sql

## Installation Supabase

Coller `sql/installation.sql` dans SQL Editor puis RUN.

## Fonctionnalités

- Connexion par identifiant sans indice visible
- Caisse employé
- Admin patron
- Gestion pizzas / recettes
- Stock
- Employés
- Livraisons
- Salaires
- Fiche imposition
- Export JPG de la fiche imposition
